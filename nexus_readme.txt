TENT PITCHER

Skyrim mod/Campfire plugin to use a power/hotkey/EasyWheelMenu to place Campfire's placeable items without opening your inventory.

:: REQUIREMENTS ::
• Skyrim Special Edition
• SKSE
• Campfire
• JContainers
• UIExtensions

:: FEATURES Template::
• Lesser powers to start placing a tent or misc item from your inventory
• Configurable hotkeys (requires MCM Helper)
• EasyWheelMenu compatibility

:: RECOMMENDED MODS ::
- MCM Helper - https://www.nexusmods.com/skyrimspecialedition/mods/53000
- EasyWheelMenu SSE Conversion - https://www.nexusmods.com/skyrimspecialedition/mods/22934
- ConsoleUtilSSE - https://www.nexusmods.com/skyrimspecialedition/mods/24858

::  KNOWN ISSUES ::
• Incomplete localization support
